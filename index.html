How to add your rom

First you have to add your rom's name to rom.xml

Then you have to create a .xml file which will contain info about your rom. The template looks like this: https://github.com/jereksel/RomManager-addictions/blob/master/template.xml.template

So,
you have to replace:
THREAD_URL with your rom url ie. http://forum.xda-developers.com/showthread.php?t=2555021
AUTHOR_NAME with your nick

Now we'll add release data
Every release starts with rom and ends with /rom
Between name and /name you have to add release name: http://oi58.tinypic.com/1on5mo.jpg
Between download /download you have to add download url
Between changelog /changelog you have to add changelog of this release

IMPORTANT NOTE: this .xml name should looks like this: YouRomNameInLowerCase.xml ie. If you add AOSPA your .xml file should be aospa.xml

Then you have to start pack.sh script, push and do pull request.

Then Travis will check if your pull request is fine. Example of bad commit: https://github.com/jereksel/RomManager-addictions/pull/6 Example of good commit: https://github.com/jereksel/RomManager-addictions/pull/7

If you are sure you've done everything fine and you still got Travis build error contact me.
